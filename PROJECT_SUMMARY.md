## 使用示例

```bash
# 基本使用
./list-project.sh

# 详细输出，包含隐藏文件，按大小倒序排列
./list-project.sh -a -f detailed -s size -r

# 输出JSON格式，限制深度为2层
./list-project.sh -f json -d 2

# 只显示Python文件，排除缓存目录
./list-project.sh -i "*.py" -e "__pycache__"

# 运行测试验证功能
./test-list-project.sh
```
